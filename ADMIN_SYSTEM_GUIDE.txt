                            TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Q: Admin can't login?
A: Verify email and password match exactly. Check if admin was created with correct key.

Q: Login redirects to wrong page?
A: Clear browser cache, verify LoggingServlet servlet path is correct.

Q: Admin Dashboard shows no users/properties?
A: Verify data files exist in WEB-INF/ folder: users.txt, properties.txt

Q: Admin key field not appearing on register page?
A: Clear browser cache, ensure JavaScript is enabled.

Q: Can't delete a user?
A: Admin cannot be deleted. Other users should have delete button.
